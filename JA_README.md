# 概要

このプロジェクトの目的は、フェイスデータをストリーミング配信するサーバーの作成です。
*フェイスデータ*: 顔の各パーツの位置情報の集まり


# Installation


```bash
$ git clone https://github.com/Cj-bc/Face-data-server
$ cd Face-dataserver
$ pipenv install
```

# 使い方


```bash
$ pipenv run python find.py
```
